# AGENTS

Этот файл задаёт правила работы Codex в текущем репозитории. **Следуй этим правилам во всех будущих промптах и изменениях кода в этом репозитории.**

## Документы (ссылки)

- `docs/antd.txt` — заметки/гайд по Ant Design и паттернам использования.
- `docs/ARCHITECTURE.md` — архитектура проекта.
- `README.md` — запуск/стек/общие договорённости.

## Вёрстка и UI

- Сначала переиспользуй существующие компоненты проекта.
- Если подходящих нет — используй компоненты Ant Design.
- Для кастомизации Ant Design в первую очередь используй тему в `src/shared/config/theme.ts`.
- Если кастомизация через тему невозможна — используй utility-классы Tailwind.

## Проверки после изменений

- После каждого изменения кода запускай `npm run lint`.
- Все найденные ошибки линтинга должны быть исправлены в рамках той же задачи.
